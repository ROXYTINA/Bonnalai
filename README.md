# 🚀 Bonnalai

**Bonnalai** is my first cross-platform full-stack project, built to explore modern software development across web and mobile platforms.

It demonstrates how a single backend can power multiple frontends (web + mobile) using a shared API and cloud database.

---

## 🌍 Project Overview

Bonnalai is a learning-focused application that connects:

- 🌐 Web Application  
- 📱 Mobile Application  
- ⚙️ Backend API  
- ☁️ Cloud Database  

The goal of this project is to understand real-world **cross-platform architecture** and full-stack integration.
